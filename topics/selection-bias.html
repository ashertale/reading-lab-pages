選樣偏差

Selection Bias

如果你拿來看的樣本不是從整體中公平地進來，那看起來再漂亮的統計也可能只是在描述篩選機制，而不是描述世界。

Read For

這一頁要把『我手上有資料』和『這份資料能代表我要推論的整體』拆開，因為很多自信結論只是忘了樣本怎麼進場。

主題類型
推論偏誤 / 樣本失真
核心問題
Biased sampling distorts what conclusions seem justified
主要判準
抽樣機制、可見樣本、遺漏樣本、推論範圍
讀完產出
能在分析與研究裡更敏感地追問樣本來源，而不是只看結果本身
樣本 篩選 失真
Reading Path 讀法總覽 問題設定 判準透鏡 心理連結 現實映射 誤讀校正 延伸提問 來源

Orientation

樣本不是中立入口，它怎麼被選進來，會直接決定你能看見什麼世界

選樣偏差比倖存者偏差更廣，它關心的是任何讓樣本進場機制歪掉的過程。

Core Tension

資料讓人有掌握感，但若樣本只代表某種容易被看見、願意回覆或被制度留住的子集，你的結論就可能被入口條件綁住。

01 先問資料怎麼來

比起先看表格，更該先看哪些人、事、事件有機會被放進表格。

02 再找缺席者

沒出現的樣本有時比出現的樣本更影響結論。

03 最後限制推論範圍

若樣本不代表整體，就別把結論說得像在講整體。

Setup

從一份看起來很完整、其實只覆蓋特定入口的資料開始

選樣偏差在使用者研究、問卷、指標、醫療資料與產品分析裡都很常見。

最小劇情

你只分析有主動回報問題的用戶，於是很容易高估某些痛點、低估那些默默流失卻不會回報的人。

真正難點

資料不是沒有價值，而是它描述的可能只是『會被你看見的人』。

樣本進場條件不均
→
可見資料偏向特定子群
→
整體推論被入口機制扭曲
Key Point

選樣偏差最有力的提醒是：你常不是在看世界，而是在看世界加上一層篩網。

Lenses

四個看樣本入口的角度

想知道結論能不能外推，第一步不是跑更多圖，而是把入口畫出來。

誰會進來

哪些對象較容易被觀察、被記錄或願意回應？

誰被漏掉

哪些人或情況幾乎沒有出現在資料裡？

篩選是否與結果相關

如果進場機制本身和你要測的現象有關，偏差通常更嚴重。

結論外推範圍

這份資料最多只能代表哪個群體？

資料來源 可能的選樣偏差 會導致什麼誤讀
自填問卷 願意填的人特別有感 把高情緒者誤當整體
回報系統 只看會回報問題的人 忽略沉默流失與無聲痛點
臨床資料 只看到院者 把就醫樣本特性誤當整體健康狀態

Psychology

為什麼看到有資料，我們就容易放心

因為資料的存在本身會給人一種完整感，讓入口偏差變得不那麼顯眼。

可視樣本太有存在感

眼前有數字時，缺席者很容易被忘掉。

表格掩蓋入口條件

分析畫面常乾淨得像樣本自然存在。

人愛把手上資料當全貌

當下有什麼，思考就容易只繞著那些內容轉。

選樣偏差不是分析細節，它決定你到底有沒有資格說那個結論。

Applications

把抽樣自覺帶回產品分析與研究

對產品與工程團隊來說，很多誤判不是算錯，而是看錯誰有機會被算進來。

使用者訪談

只找活躍用戶，通常會低估上手失敗與流失者的問題。

品質指標

只看成功跑完的測試樣本，可能忽略根本啟不起來的那批。

客訴數據

會抱怨的不等於全部受影響的人，更不等於最重要的人。

  • 每次看到資料結論前，先問一句：誰有機會進這份資料，誰沒有。
  • 如果無法改善抽樣，至少要縮小結論口氣，不要把子樣本講成整體真相。

Misreadings

三個常見誤讀

選樣偏差不是只有學術研究才要在意，任何資料決策都在碰它。

樣本夠大就沒事 校正

樣本很大但入口偏，還是會穩定地偏。

只要有數據就比較客觀 校正

客觀不只看量，還看進樣機制。

這跟倖存者偏差一樣 校正

倖存者偏差是其中一種，選樣偏差範圍更廣。

Questions

這份資料是在描述世界，還是在描述誰比較容易被放進資料裡

下面兩個問題很適合放在分析開頭。

哪些重要群體幾乎沒有機會出現在這份資料中

這往往是最該先寫在結論旁邊的限制。

進場條件本身是否和我要測的結果有關

若有，偏差通常不是小修能補。

延伸路線

延伸閱讀可接 sampling, survivorship bias 與 external validity。
