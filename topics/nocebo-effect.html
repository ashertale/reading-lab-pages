Core Tension

當人預期某件事會帶來副作用、疼痛或惡化時，這份預期本身就可能加劇主觀甚至生理上的負面反應。

02 再看 身體反應 被怎麼剪掉

Setup

哪一步讓 負向預期 把判斷路徑偷窄了

把情境縮到一次普通判斷後，負向預期 會怎麼替你省事、又怎麼順手砍掉 身體反應，都會清楚很多。

最小劇情

病患、使用者或團隊成員只要先被灌進不良期待，後續的疼痛、疲勞與不適就更容易被放大與讀成證據。

Key Point

重要的不是背出 反安慰劑效應 這個名字，而是當 負向預期 正在替你省事時，還能意識到它同時剪掉了哪些必要資訊。

Psychology

反安慰劑效應 為什麼看起來那麼像正常判斷

反安慰劑效應 一再出現，通常不是因為人不努力，而是因為 負向預期 真的能幫我們在壓力下更快做完判斷。

先擔心再找證據

先擔心再找證據 之所以難纏，是因為它剛好踩在大腦最想省時間的地方，所以看起來特別像自然反應。

身體訊號被重新詮釋

身體訊號被重新詮釋 很常不是因為你不知道，而是因為當下那條省力路真的太順手、太像合理判斷。

壞預期比好預期更黏

大腦遇到 壞預期比好預期更黏 時，通常不是故意犯錯，而是先用最快的近路把世界壓成可處理大小。

看清 反安慰劑效應 的價值，在於你知道 負向預期 為何好用，也知道它何時開始把 身體反應 偷偷剪掉。
