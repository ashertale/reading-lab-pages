Semmelweis Reflex

當新證據直接威脅到既有做法、身份或世界觀時，人和組織可能會本能地先排斥訊息，而不是先檢查訊息。

Read For

這一頁要幫你把『這個想法不舒服』和『這個想法不成立』拆開，因為組織常把前者直接誤當後者。

Orientation

最難接受的新證據，常不是最弱，而是最傷自尊

塞麥爾維斯反射抓到的是一種保衛既有認知與制度的本能，不是單純的理性反駁。

Core Tension

新證據可能真的有問題，但如果組織在檢查前就先把它視為對秩序、專業或權威的冒犯，學習就會在門口被擋掉。

01 先辨認威脅感

有時被拒絕的不是證據內容，而是它暗示既有做法可能一直有問題。

02 再分開人與說法

若新訊息一出現就等於誰要丟臉，理性檢驗很難啟動。

03 最後設計試驗

把全盤推翻改成小範圍驗證，常比正面硬撞更能讓新證據活下來。

Setup

從一個令人不舒服的新數據開始

這個概念的現代版本，常不是公開怒斥，而是安靜地不想理它。

最小劇情

量測結果反覆顯示某個長年信任的流程會在特定條件下失效，但團隊第一反應是懷疑量測治具、懷疑提報者、懷疑情境太特例。

真正難點

不是任何新說法都值得立刻相信，而是你是否願意先給它一個被檢驗的機會。

新證據威脅既有做法
→

→

Key Point

塞麥爾維斯反射真正提醒你的，是『不舒服』不能直接當成『不成立』的理由。

Lenses

四個看證據排拒的角度

當新訊息一進場就被排斥，常見原因大概落在這四類。

身份威脅

若接受新證據等於承認自己長期判斷錯誤，防衛會特別強。

制度摩擦

有些證據不是不好，而是接受它會連帶要求流程、分工或責任重寫。

敘事慣性

一個說法被講久了，就算新數據出現，舊故事仍有巨大的心理黏著力。

驗證成本

若檢驗新證據需要額外時間與資源，組織更可能先用否認節省成本。

面對新訊息時 看起來像 其實可能是
先質疑提出者 在做品質把關 其實是用身份攻擊跳過內容檢驗
一直說情境太特例 很嚴謹 其實是在避免證據進入主敘事
要求完美證明 標準很高 其實把驗證門檻設得高到沒有新說法能進來
不斷延後試驗 資源有限 其實是把不舒服問題冷凍掉

Psychology

為什麼打臉感會先於求真心態

因為新證據不只是在挑戰事實，也常在挑戰我們的專業形象、責任紀錄與群體秩序。

自我保護很快

一旦感到『接受它等於承認我之前錯』，大腦會先站穩再說。

群體共識有黏著力

當大多數人都已經採用同一說法，偏離它會有社會成本。

改變意味著重工

有時排拒不是因為不信，而是因為太清楚一旦接受就要重做很多事。

這個概念最有用的地方，是讓你看見新證據被擋下時，阻力不一定來自證據品質本身。

Applications

把證據入口設計回團隊流程

在工程與實驗工作裡，塞麥爾維斯反射最常出現在大家已經對某套流程投入太深的時候。

量測與驗證

當 field return 與 lab data 長期對不上時，團隊可能先保衛既有 lab 模型，而不是先追哪裡沒被量到。

Legacy 假設

某個設計規則傳了很多年，新的 failure evidence 進來時，很容易先被說成『只是操作不標準』。

安全與流程改革

越是要改寫責任邊界的證據，越可能在制度入口就被卡住。

  • 讓新證據能先進入小範圍驗證，而不是一開始就要和整套舊制度正面對撞。
  • 檢討拒絕證據的理由時，刻意分開『證據本身薄弱』和『接受它代價太高』。

Misreadings

三個會把它變成口水戰的誤讀

塞麥爾維斯反射不是讓你把所有不同意者都罵成保守派。

別人不同意我，就是 Semmelweis reflex 校正

不對。新證據本來就需要檢驗，重點在於是否在檢驗前就被情緒封殺。

只要不舒服就應該接受新說法 校正

不舒服不是充分條件，仍要看資料、方法與可重現性。

這只發生在醫學史 校正

任何高承諾、高身份連結的制度，都可能長出相同防衛。

Questions

我現在是在檢驗證據，還是在保護自己

下面兩個問題適合在面對不舒服的新資訊時自問。

如果這個證據是真的，我最不想承受的是什麼

是責任重分、流程重做，還是承認我過去判斷有盲點？

我有沒有給它一個可驗證的入口

還是只是因為它會動搖現狀，就直接把它留在門外？
