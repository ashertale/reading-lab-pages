假兩難

False Dilemma

當問題被壓縮成『不是 A 就是 B』時，討論常會突然變簡單，但也可能把真正可行的第三條路、漸進方案或條件式答案一起消失。

Read For

這一頁要把『這題就是二選一』和『其實是被說成只剩二選一』拆開，因為很多僵局是選項被提前砍掉的結果。

主題類型
推理偏誤 / 選項壓縮
核心問題
Presenting a situation as if only two options exist when more are available

Orientation

把複雜問題壓成二選一，常會同時帶來清楚感與盲點

假兩難之所以有吸引力，是因為它讓決策像考選擇題一樣俐落；問題在於現實常不只兩格。

Core Tension

壓縮選項能加速表態，但若可行空間本來就更寬，二分法只會讓團隊在錯框架裡互撞。

01 先問還有沒有第三類

很多題目其實有混合、階段式或條件式答案。

02 再分清描述與壓縮

有時說成兩邊只是為了簡化溝通，不代表現實只有兩邊。

03 最後把漏掉的代價算進來

若二分法讓重要選項失語，決策品質就會被一起壓扁。

Setup

從一句『不是現在上就是永遠不上』開始

假兩難在策略討論、管理溝通與輿論辯論中特別常見。

最小劇情

有人把架構決策說成『要嘛全面重寫，要嘛什麼都別碰』，於是分段改造、局部抽離與試點路線全被擠掉。

真正難點

一旦大家接受這個二分框架，後面就算再聰明也只能在被切窄的地板上跑。

複雜問題被壓成兩端
→
其他選項失語
→
討論在假二選一中升溫
Key Point

假兩難最需要防的是：在表態變容易時，思考空間也被一起關掉。

Lenses

四個看選項壓縮的角度

不是所有二分都錯，但當二分讓關鍵中間地帶消失時，就值得停下來重畫地圖。

遺漏選項

是否存在漸進方案、試點方案或不同維度的解法？

維度混疊

是不是把本可分開處理的問題硬綁成同一條軸？

話術速度

二分法是否主要用來迫使快速站隊？

成本對照

被排除的選項，其實是否才最符合當下資源與風險條件？

二分說法 漏掉什麼 可能後果
全上或全不上 試點、分階段導入 把可逆路徑一起關掉
快或穩只能選一個 重排範圍、切分風險 把治理問題誤寫成天生對立
支持或反對 條件支持、局部支持 對話更極化

Psychology

為什麼二分法這麼誘人

因為它能快速降低複雜度，讓人立刻知道自己站哪裡。

站隊比建模快

二選一比把空間攤開來看省很多認知成本。

衝突敘事更有張力

兩端對立比多元中間地帶更容易引發情緒與動員。

模糊中間比較難說

很多中間方案需要更多條件說明，因此常在口語場域吃虧。

假兩難不是說不能在最後二選一，而是說別太早把世界壓成只剩兩格。

Applications

把選項空間重新打開帶回決策

對工程與產品團隊來說，假兩難最常讓可逆、漸進、混合式解法提前失語。

架構演進

全面重寫與完全不動之間，常還有抽取邊界、局部服務化與 adapter 過渡。

流程治理

快與穩不是只能二選一，很多時候可以透過分層 gate 重畫 tradeoff。

產品策略

功能不是全開全關，也可以限量、灰度或條件開放。

  • 只要有人說『不是 A 就是 B』，就值得追問至少兩個被排除的第三選項。
  • 先把可逆方案找出來，常能讓假兩難鬆開一大半。

Misreadings

三個常見誤讀

假兩難不是說任何二選一都錯，而是說二選一若遮掉現實可行空間就有問題。

有時最後真的要二選一，所以這不重要 校正

最後也許要選，但在那之前仍不該省略其他可能。

只在辯論術才有 校正

團隊會議、產品路線與技術決策也常出現。

中間選項一定比較好 校正

不一定，重點是讓它們有機會被看見和比較。

Questions

我是在面對真二選一，還是被提早關進二選一

下面兩個問題很適合放在僵局討論裡。

如果不能選 A 或 B，還有哪些第三路徑其實存在

這個問題常能立刻打開空間。

這兩端是否其實混了不同維度，才看起來像只能站隊

拆維度後常會出現新解。
