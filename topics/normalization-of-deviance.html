Orientation

如果某做法明明違規，為什麼只因為短期沒爆，就能一路被養成新常態

這頁真正要處理的，不是你喜不喜歡 偏差常態化，而是遇到 安全流程 或 值班作法 時，哪些前提若沒先寫出來，討論就會失焦。

Core Tension

一件本來明顯偏離規範的做法，只要反覆沒有立刻出事，就可能慢慢被整個群體當成正常。

01 先把 偏差累積 拉回起點

很多爭論表面在談結論，其實一開始就把 偏差累積 當成默認前提。

02 再問 沒出事錯覺 是誰在付

現場最常漏掉的不是立場，而是 沒出事錯覺 到底由誰承擔、何時結帳。

03 最後確認 文化默認 誰能檢查

真正把事情做完的，往往不是表態，而是把 文化默認 翻成可被檢查的動作。

Setup

把 偏差常態化 放進一次真實判斷，看誰先該交代

把討論縮到一次具體決策，偏差累積 是否先被交代、沒出事錯覺 是否被偷省、文化默認 是否有驗收方式，就會一眼看出來。

最小劇情

團隊為了趕進度開始偶爾省略某道檢查，幾次都沒出事後，這個例外就慢慢被說成『其實平常都這樣』。

真正難點

如果某做法明明違規，為什麼只因為短期沒爆，就能一路被養成新常態

有人先把 偏差累積 當既成前提
→
真正要付的 沒出事錯覺 被留到後面
→
收尾時才發現 文化默認 根本沒有操作方式
Key Point

把 偏差常態化 用好，重點不在記住一句格言，而在於知道 偏差累積 要先說、沒出事錯覺 要有人扛、文化默認 要能驗收。

Lenses

先把 例外變常規、無事故不等於安全 與 文化默認比規章更有力 分開看

如果不把 例外變常規、無事故不等於安全 與 文化默認比規章更有力 拆開，你很容易以為大家在討論同一條原則，實際上是在搶不同的判準。

例外變常規

順著 例外變常規 讀，你會發現爭點不在口號，而在 偏差累積 是否真的被變成判準。

無事故不等於安全

把焦點放在 無事故不等於安全 時，你其實是在說：沒出事錯覺 應該先被寫清楚，否則後面的結論都站不穩。

文化默認比規章更有力

文化默認比規章更有力 這條線在乎的，不是誰比較有氣勢，而是 文化默認 一旦沒被交代，現場會怎麼開始失真。

Psychology

為什麼人一忙起來就最容易把 偏差常態化 用歪

這類原則常被用歪，不是因為內容太難，而是因為把它說成一句漂亮話，比真的交代 偏差累積、沒出事錯覺 和 文化默認 容易太多。

沒出事就以為可接受

碰到 沒出事就以為可接受 時，最危險的是大家以為原則已經被引用，實際上 偏差累積 還沒被具體說明。

例外一次比一次容易

例外一次比一次容易 會讓人覺得只要把話說得夠完整，就不必再補 沒出事錯覺 的證據或代價。

成功掩蓋危險

成功掩蓋危險 常不是純粹錯誤，而是一種討論捷徑：先把自己放在合理的一邊，再晚點處理 文化默認。

一條原則會變空話，通常不是因為字太抽象，而是因為 文化默認 從來沒有被翻成現場動作。

Applications

回到 安全流程、值班作法 與 現場 workaround 治理

真正有差的，不是會議上有沒有人引用 偏差常態化，而是 安全流程 和 值班作法 裡的決策流程有沒有因此變得更可檢查。

安全流程

在 安全流程 這種場景，偏差常態化 幫你的不是贏辯論，而是把 偏差累積 先釘在桌面上，避免事後才補。

值班作法

值班作法 最需要 偏差常態化 的時候，通常就是大家都急著往前走，卻還沒把 沒出事錯覺 寫成共識。

現場 workaround 治理

只要到了 現場 workaround 治理，你就能看出 文化默認 究竟是被真的承擔，還是只是被口頭提過。

Misreadings

三個常見誤讀

這類頁面最常被讀歪的地方，不是概念太抽象，而是大家太快把 偏差累積 當成價值宣示，忘了它還得連著 沒出事錯覺 和 文化默認 一起成立。

偏差常態化 只是好聽口號 校正

真正難的是把 偏差常態化 翻成一組可檢查的判準，而不是把它當成結論本身。

只要背了 偏差常態化 就能直接套用 校正

原則若不落回 偏差累積、沒出事錯覺 和代價分配，通常會在現場變成誤用。

偏差常態化 只適合高層決策 校正

其實從 code review 到 安全與組織文化 的日常判斷，都會碰到同樣的壓力。

Questions

把 偏差累積、沒出事錯覺 與 文化默認 變成會議上的追問

下面兩個問題不是要你再複述一次原則，而是逼 安全流程、值班作法 這類場景裡的人把空白前提補完。

如果 安全流程 現在就要拍板

誰要先把 偏差累積 說清楚？沒出事錯覺 由誰承擔？若 文化默認 無法驗收，你還願意讓決定往前走嗎？

如果團隊說『偏差常態化 不是大家都同意嗎』

怎樣才算真的用了 偏差常態化，而不是只借它替既有結論加上道德光澤？

延伸路線

把 Safety Culture 一起讀，會更清楚 偏差常態化 在別的框架裡是怎麼被重新安排的。
