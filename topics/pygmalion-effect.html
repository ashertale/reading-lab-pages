Pygmalion Effect

他人對你的期待不只是評語，還可能改變他們給你的機會、回饋與耐心，最後影響你的表現。

Orientation

期待會改變待遇；待遇再改變表現

皮格馬利翁效應提醒我們，高期待可能透過更多機會、更細回饋和更長耐心，幫助人真的表現更好。

Core Tension

期待能啟動成長，但若期待分配不公平，它也會把早期印象變成後來的成績差距。

01 把期待變成行為

不要只說相信誰，而要看誰得到更難任務、更好回饋和更多等待。

02 檢查機會分配

若高期待者得到更多練習，表現差距就不能只解釋成能力差距。

03 讓低期待可被修正

早期錯判需要有證據能推翻，否則期待會鎖住發展。

Setup

被看好的新人，常先得到更像成長的環境

最小場景是主管相信某人潛力高，於是自然給他更多教學、耐心與高價值任務。

最小劇情

兩位新人加入團隊。主管覺得其中一位反應快，便讓他參與架構討論並提供細緻回饋；另一位只拿到瑣碎修 bug。半年後，第一位確實成長更快。

真正難點

結果可能是真的表現差距，但差距的一部分來自期待提前改變了資源和練習場。

初始期待形成
→

→
表現差距反過來證明期待
Key Point

皮格馬利翁效應的校正，是檢查期待如何變成具體待遇，而不是只問誰有天分。

Lenses

追蹤期待如何進入日常互動

期待若沒有行為通道，就只是想法；真正影響結果的是它改變了什麼互動。

任務難度

誰得到 stretch task，誰只被安排低風險雜務？

回饋密度

誰得到具體修正與二次機會，誰只得到簡短評語？

容錯耐心

高期待者失誤時是否被視為學習，低期待者失誤時是否被視為證據？

Psychology

人會尋找符合期待的證據，也會用期待調整投入

期待讓管理者更容易看見某些訊號，也更願意投資某些人；被期待者也會讀到這些差異。

確認線索被放大

主管容易注意到符合初始印象的表現，忽略反向證據。

投入變得不對稱

被看好的人獲得更多教學和等待，低期待者則較少得到修正機會。

被期待者調整自我概念

人感受到他人相信自己時，可能更敢嘗試；反之則更快退縮。

公平的期待不是假裝每個人一樣，而是讓期待能被行為證據更新。

Applications

把期待迴路設計成可審計

Mentoring、績效管理和教育訓練都需要避免初始期待鎖死發展。

Mentoring

紀錄每位成員得到的高價值任務、回饋次數與二次機會，而不是只記最後表現。

績效管理

評估表現時同時檢查資源和機會分配，避免把環境差異全算成個人能力。

教育訓練

讓所有學員都有機會接觸挑戰題與具體回饋，再比較學習曲線。

Misreadings

不是期待越高越好

高期待若沒有支援，會變成壓力；低期待若不能更新，會變成封鎖。

把它當成正向思考 校正

真正起作用的是期待改變了互動、機會和回饋，不是單純相信。

以為所有差距都由期待造成 校正

能力、經驗與努力仍存在；要校正的是不公平期待迴路。

只提高標準不給支援 校正

高標準若沒有資源、回饋和時間，可能只是更大的壓力。

Questions

檢查期待變成了哪些待遇

我給誰更多成長型任務？

如果高期待者拿到更多練習，後續表現比較就要納入這個差異。

哪些低期待可以被什麼證據推翻？

若沒有更新條件，期待就會變成自我實現的標籤。

延伸路線

可接著讀 self-fulfilling prophecy，理解信念如何透過行為安排創造自己的證據。
