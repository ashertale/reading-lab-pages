貨物崇拜科學

Cargo Cult Science

貨物崇拜科學批評的是模仿科學外形卻缺少自我懷疑：儀器、統計和術語都在，最關鍵的反駁精神卻不在。

Read For

辨認方法外觀、選擇性報告、確認偏誤和研究誠實如何影響知識可信度。

Orientation

穿上白袍不等於願意讓自己的想法受傷

貨物崇拜科學指向一種危險：保留科學的儀式，卻移除科學最痛的部分，也就是主動尋找自己可能錯在哪。

Core Tension

研究者或團隊可能照做實驗、畫圖表、報 p-value，卻只呈現支持敘事的結果，讓方法成為裝飾。

01 尋找反證行為

看研究者是否主動測試最可能推翻自己的條件。

02 要求完整報告

負面結果、失敗樣本和排除規則都應可追蹤。

03 分開儀式與態度

形式符合流程，不代表推理真的開放被修正。

Setup

一份漂亮 benchmark 報告可能只展示模型最會的題型

最小場景是一個評測流程看似嚴謹，但測試選擇已經偏向成功。

最小劇情

團隊展示新模型在十個 demo 任務上大幅領先。追問後發現，原本測了三十個任務，失敗和不穩定案例被視為不具代表性而移除。

真正難點

外觀看起來有數據，但數據生成與篩選過程讓結論失去可反駁性。

科學形式被保留
→
反例被濾掉
→
結論變得像廣告
Key Point

貨物崇拜科學要求檢查研究者是否有誠實揭露自己不希望讀者看到的部分。

Lenses

看流程是否保留讓自己輸的機會

科學性不只在使用工具，也在願意讓工具回擊自己。

事前承諾

評測指標、樣本和排除規則是否事前定義？

負面材料

失敗案例是否被呈現並解釋？

外部重做

他人是否能用相同資料和步驟檢查結果？

Psychology

人可以真心相信自己誠實，仍然把不舒服的資料推到旁邊

偏誤不一定來自造假；常來自不願看見會破壞故事的結果。

敘事保護

漂亮故事讓零散反例看起來像噪音。

聲望壓力

研究或產品成功會帶來資源和地位。

方法迷信

工具越複雜，越容易讓人忘記問基本問題。

一份可信報告應該讓你看見它差點失敗的地方。

Applications

把反例與失敗納入正式輸出

AI 評測、產品分析和研究報告都需要保留負面證據。

AI benchmark

報告所有任務、排除規則和 variance，而非只列最佳 demo。

產品實驗

同時呈現主要指標、保護指標與不利切片。

研究流程

使用 preregistration 或分析計畫降低事後篩選。

Misreadings

貨物崇拜科學不是反科學

它是對科學形式被空洞化的批評，重點在恢復嚴格自我檢查。

把它當攻擊儀式 校正

流程和統計很重要，只是不能取代誠實。

只指責別人 校正

任何團隊在壓力下都可能選擇性呈現。

把失敗當羞恥 校正

負面結果是校正知識的材料。

Questions

問報告如何處理不支持自己的資料

哪些結果如果出現會推翻這個結論？

看報告是否真的保留反證條件。

失敗案例和排除樣本在哪裡？

沒有負面材料的報告需要更高警覺。

延伸路線

可接著讀 scientific integrity，理解透明、可重現和負面結果如何支撐可信研究。
