框架效應

Framing Effect

同樣一組事實，只因為被包裝成收益、損失、成功率或失敗率，人就可能做出完全不同的決策。

Read For

這一頁要把『只是換個說法』和『其實已經改變你怎麼感受風險』拆開，因為很多決策偏移都不是證據變了，而是語言先塑形了。

主題類型
決策偏誤 / 表述影響
核心問題
Choices shift when equivalent information is presented differently
主要判準
等價資訊、收益損失框架、風險感受、語言暗示
讀完產出
能在規格、簡報與風險溝通裡看見表述方式如何改變選擇

Orientation

資訊沒有變，選擇卻變了，差別常只在你先被帶進哪個框裡

框架效應提醒你：人的選擇不只受內容影響，也受內容被放進哪種語意容器影響。

Core Tension

表述總得選角度，但一旦某個角度特別凸顯損失或收益，原本應該等價的選項就可能被感受成完全不同。

01 先找等價是否真的等價

如果只是文字包裝不同，卻得到不同反應，就可能是框架效應在作用。

02 再問哪種感受被放大

是風險、損失、成功希望，還是道德感？

03 最後做雙框檢查

重要決策最好同時用正向與負向表述看一次。

Setup

從一個其實沒變的選項，只因說法不同就換票開始

框架效應特別常出現在醫療、產品預設、風險簡報與管理溝通。

最小劇情

把同一個方案說成『90% 成功率』和『10% 失敗率』，即使數學等價，接受度仍可能顯著不同。

真正難點

人不是只在算機率，也在感受語言幫自己點亮了哪一種心理場景。

同一資訊換不同表述
→
情緒與風險感受被改寫
→
原本等價的選項得到不同偏好
Key Point

框架效應最有力的提醒是：溝通方式本身，就是決策介面的組成部分。

Lenses

四個看表述偏移的角度

不是所有 framing 都是不誠實操控，但重要選擇至少該知道自己被哪種語氣推了一把。

收益或損失

同一方案是被說成保住多少，還是會失去多少？

絕對值或比例

小基數的大比例和大基數的小比例，感覺很不一樣。

責任語氣

這是被描述成主動作為，還是被描述成避免犯錯？

時間視角

強調短期痛感或長期回報，也會改變風險接受度。

表述方式 感受被推向哪裡 選擇可能如何偏
成功率 樂觀與保住收益 較願意接受方案
失敗率 損失與風險顯化 較容易保守或拒絕
節省成本 看見效率收益 較願意支持變更

Psychology

為什麼換句話說真的會換掉決策

因為人不是只回應事實，也會回應事實被放進的情境感與情緒方向。

語言會先定義情緒入口

先看見損失還是先看見收益，會改變整個評估姿態。

等價不等於等感

數學對等不會自動轉成心理對等。

表述像隱形預設

它不需要改變資料，就能改變你第一眼把重心放在哪。

框架效應不是說語言比事實重要，而是說語言會決定你怎麼靠近事實。

Applications

把雙框檢查帶回規格、簡報與產品文案

對產品、工程和管理來說，框架效應最常讓團隊在以為自己很理性時，其實被文案順手帶偏。

風險簡報

把 rollback 成本講成延誤，和講成避免事故，會導向不同決策氛圍。

產品選項

訂閱方案若以省下多少表述，和以會損失什麼表述，轉換率常不同。

專案提案

講變更的好處與講不變的代價，對管理層接受度的影響可能完全不同。

  • 重要決策請至少做一次雙向 framing，把同一方案用收益與損失各講一遍。
  • 若某提案只有靠單一框架才看起來合理，通常就值得再多檢查。

Misreadings

三個常見誤讀

框架效應不是說所有說法都在操控，而是說表述本來就會塑形。

只要資料正確就不受 framing 影響 校正

同資料仍可能因表述方式而改變選擇。

所以不能用任何有角度的語言 校正

不可能完全無框架，重點是要自覺並做對照。

只有行銷才會用 framing 校正

醫療、法規、工程提案與事故溝通都很常見。

Questions

如果我把同一件事換另一種說法，這個決定還成立嗎

這個選項若改成相反框架表述，我還會一樣偏好嗎

能幫你看見選擇是否過度依賴語言包裝。

我現在的文案是在幫人看清事實，還是在偷偷指定感受方向

兩者有時會混在一起。
