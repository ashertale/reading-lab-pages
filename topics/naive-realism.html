天真實在論

Naive Realism

我們很容易相信自己看到的是事實本身，而與自己不同的人不是資訊不足，就是被偏見或立場扭曲了。

Read For

這一頁要把『我只是客觀看事』和『其實我也站在某個視角裡』拆開，因為很多對話之所以越講越硬，就是雙方都把自己當成現實代理人。

主題類型
社會認知 / 視角盲區
核心問題
Believing we see the world objectively while others are biased
主要判準
視角自信、他者歸因、資訊選取、對話空間
讀完產出
能更早察覺自己把觀點當事實、把分歧當他人偏誤的時刻
我看見 就是事實 他有偏

Orientation

最難鬆動的偏見，常是相信自己根本沒有偏見

天真實在論不是說世界沒有事實，而是說人太容易把自己的觀看位置忘掉。

Core Tension

相信自己客觀能帶來行動力，但若把分歧都讀成他人有問題，就會讓真正的資料與判準對話越來越少。

01 先承認自己也在取景

你看到的從來不是全部現實，而是被位置、語言與關注點切過的版本。

02 再把判準顯性化

與其說『這不是很客觀嗎』，不如說自己在用哪些證據與優先序。

03 最後容許分歧不等於惡意

不同判斷有時不是誰更理性，而是站在不同風險與目標面。

Setup

從一場雙方都覺得自己只是在講事實的爭論開始

天真實在論最常讓衝突升高，因為它把分歧直接翻譯成對方有問題。

最小劇情

工程覺得某設計風險顯而易見，產品覺得時程壓力更顯而易見；雙方都認為自己只是客觀描述現況。

真正難點

當每個人都自認在講事實，對方的不同意就更容易被讀成不理性。

把自己視角當客觀現實
→
把分歧讀成他人偏誤
→
對話從比判準變成比誰比較清醒
Key Point

天真實在論最危險的地方，是它會讓你連自己在選擇觀看方式都沒意識到。

Lenses

四個看視角盲區的角度

真正有幫助的不是放棄立場，而是知道立場本來就存在。

證據選取

你特別看重哪些訊號，忽略哪些訊號？

優先序

你在保什麼：速度、可靠性、公平性、成本還是控制感？

角色位置

你所承擔的責任是否正在塑造你覺得什麼最明顯？

他者解釋

你是否太快把分歧歸因為對方愚蠢、偏頗或別有用心？

說法 聽起來像 更誠實的版本
這不是事實嗎 我沒有立場 我現在把某些證據與風險看得特別重
只有一個合理答案 分歧者不夠理性 不同角色可能在優先不同代價
對方只是情緒化 我才是客觀 也許雙方都在用不同框架選資料

Psychology

為什麼人很難感覺到自己也有視角

因為自己的觀看方式總是從內部被體驗成透明的，很像直接看到世界本身。

內部視角最滑順

自己的判斷過程不會像外人那樣顯得陌生。

他者差異最顯眼

別人和你不同的地方，比你和自己相同的預設更容易被看見。

客觀感很能保護自我

把自己視為中立觀察者，比承認自己也在取捨來得舒服。

天真實在論不是要你放棄真偽判斷，而是要求你先誠實面對自己也在取景。

Applications

把視角自覺帶回衝突與決策

對工程與跨部門合作來說，很多僵局不是沒有資料，而是彼此都把自己那一側的資料當成整個世界。

架構爭論

可靠性派和速度派都可能覺得自己只是在講常識。

事故回顧

不同角色會把同一事件讀成完全不同的關鍵節點。

產品取捨

使用者體驗、法規風險與工程成本常各自看見不同的『最明顯事實』。

  • 當對話開始出現『這不是很明顯嗎』，通常就是把判準攤開來談的時機。
  • 若想降低衝突強度，先說明自己在保什麼與怕什麼，通常比先爭誰更客觀有效。

Misreadings

三個常見誤讀

天真實在論不是相對主義，而是提醒你客觀主張也需要交代取景位置。

所以沒有客觀事實 校正

不是。重點是我們接近事實時，常帶著自己的切片與語境。

只要承認視角就不能做決策 校正

相反地，承認視角能讓決策更誠實。

這只在政治爭論才有 校正

產品、工程與團隊協作裡同樣常見。

Questions

我現在是在描述世界，還是在描述我站在哪裡看世界

下面兩個問題很適合放在分歧升高時。

我現在認為最明顯的事實，是否其實仰賴某些未說出的優先序

一旦說出來，對話通常會乾淨很多。

如果對方不是笨也不是壞，他還可能根據什麼資料得出不同結論

這能打開更有用的分歧空間。
