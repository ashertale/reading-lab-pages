單純曝光效應

Mere Exposure Effect

很多東西不是因為本身更好才被喜歡，而是因為它比較常被看見、比較熟，於是大腦對它比較放鬆。

Read For

這一頁要把『它真的更好』和『只是我比較熟所以比較不排斥』拆開，因為熟悉感常默默在替偏好加分。

主題類型
偏好心理 / 熟悉增益
核心問題
Repeated exposure can increase liking or comfort

Orientation

常看到的東西，常不知不覺就變得比較可接受

單純曝光效應提醒你：偏好有時不只是品質判斷，也有熟悉帶來的放鬆感。

Core Tension

熟悉能降低認知負擔與不安，但若把熟悉誤當成品質，就可能錯把慣性當優越。

01 先辨認熟悉紅利

偏好形成時，先問自己是不是只是比較常接觸。

02 再區分舒適與品質

比較順眼不代表比較對，也不代表更適合。

03 最後善用但別濫用

設計上可以利用熟悉感降低摩擦，但不該拿它替真正價值補洞。

Setup

從一個大家越看越順眼、卻未必真的越好的方案開始

單純曝光效應在 UI、品牌、用字、團隊工具與人際印象裡都很常出現。

最小劇情

團隊越看熟的 dashboard 越覺得合理，即使它其實藏了不少資訊結構問題；新方案雖更清楚，第一眼卻覺得怪。

真正難點

熟悉會讓人感到安全，於是偏好很容易在你還沒做完整比較前就先形成。

反覆接觸同一刺激
→
熟悉感上升、戒心下降
→
偏好與信任被默默推高
Key Point

單純曝光效應最有用的提醒是：舒服和優秀常不是同一件事。

Psychology

為什麼熟悉會讓人比較放鬆

因為大腦面對可預期刺激時，處理成本較低，也就更容易把它感成安全或可接受。

處理流暢會誤導評價

越容易處理的東西，越容易被感覺成比較好。

陌生感像小風險

即使沒明確壞處，陌生也常被體驗成輕微威脅。

重複會慢慢洗掉戒心

只要沒明顯負面事件，曝光本身就能提高接受度。

單純曝光效應不是說熟悉一定誤導，而是提醒熟悉本身就在投票。

Applications

把熟悉感放回設計與選型判斷

對產品與工程團隊來說，很多『這個比較順』的評論，背後其實混著熟悉紅利。

介面改版

使用者剛開始的不適不一定代表設計更差，也可能只是曝光量還不夠。

工具選型

團隊熟悉某套工具後，會更容易高估它、低估新工具。

文案與命名

反覆曝光會提升接受度，因此第一次可讀性和長期熟悉效果要一起看。

  • 做設計比較時，別把『我比較熟』誤當成『它真的更好』。
  • 若要推新方案，可以先增加低風險曝光，讓抗拒別只來自陌生感。

Misreadings

三個常見誤讀

單純曝光效應不是說重複洗版就一定有效，而是說熟悉本來就在塑形偏好。

只要多看就會愛上 校正

若刺激本身帶強烈負面經驗，曝光不一定能救。

熟悉就等於好用 校正

熟悉只降低負擔，不保證功能與結構更好。

這只跟行銷有關 校正

產品、團隊工具、文件與人際判斷都常見。
