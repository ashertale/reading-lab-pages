誘餌效應

Decoy Effect

一個看似沒人會選的壞選項，仍可能改變偏好；它的任務不是被買走，而是讓某個方案顯得更划算。

Orientation

壞選項不一定是雜訊，它可能是用來改變比較基準的工具

誘餌效應提醒我們，偏好不是只由兩個主要選項決定；旁邊那個明顯較差的第三者也會重新安排注意力。

Core Tension

比較能幫人判斷價值，但被設計過的比較會把需求問題改寫成『哪個看起來比較賺』。

01 先找被支配選項

誘餌常在價格、容量或功能上明顯輸給目標方案，卻和它足夠接近。

02 再看偏好流向

如果第三選項讓某方案突然顯得划算，偏好可能是被相對比較推動。

03 最後回到需求

真正要買的是需求匹配，不是比較表裡的局部勝利。

Setup

沒人想買的方案，可能是最會影響銷售的方案

最小場景是第三個選項在絕對上不吸引人，卻在相對上替目標方案製造優勢。

最小劇情

定價頁加入一個容量少、價格卻接近 Pro 的方案。使用者很少選它，但它讓 Pro 看起來比 Basic 划算許多。

真正難點

誘餌選項不是提供需求匹配，而是改變比較方向；如果不檢查，決策者會以為偏好自然形成。

→
目標方案相對變亮
→
原本偏好被重新導向
Key Point

誘餌效應的校正，是把選項放回需求表，而不是只看誰在相對比較中勝出。

Lenses

辨認誘餌，要看它服務誰

一個選項若自己很難成立，卻讓另一個選項更容易成立，它就值得被檢查。

支配關係

誘餌通常在多數維度都輸給目標方案，卻價格或條件接近，讓目標顯得划算。

注意方向

它會把注意力從真正需求移到某個容易比較的維度，例如容量、速度或折扣。

偏好移動

拿掉誘餌後，如果使用者在原兩個方案間的偏好大幅改變，選擇可能被架構牽引。

Psychology

人在複雜選擇裡喜歡找局部勝利

直接判斷價值很難，相對比較很容易；誘餌就是把容易比較的局部勝利送到眼前。

局部支配帶來確定感

看到 A 明顯勝過 C，人會把這種確定感帶到 A 和 B 的選擇裡。

需求被比較噪音覆蓋

越多方案並列，人越容易用最醒目的差異做決策，而不是回到原需求。

划算感替價值背書

看起來划算會讓人忘記自己是否真的需要多出來的功能。

校正誘餌效應，就是把被設計出來的相對優勢拆回需求、成本與使用情境。

Applications

誘餌常出現在定價頁，也會出現在技術選型表

任何比較表只要包含明顯較差但位置巧妙的選項，都可能在塑造偏好。

定價設計

檢查每個方案是否有獨立受眾；若某方案只為襯托另一方案存在，就要標明這是選擇架構風險。

採購比較

供應商方案表裡的低配高價項目，可能讓中高階方案看起來特別合理。

架構選擇

拿一個明顯不合適的極端方案作比較，會讓目標架構不經需求檢查就獲勝。

Misreadings

誘餌不是所有第三選項

第三選項可以提供真實需求分段；問題是它是否主要靠輸給某方案來發揮作用。

把分級都視為操控 校正

合理分級服務不同使用者；誘餌問題在於選項缺乏獨立價值。

只看銷售結果 校正

如果目標方案銷售變好，仍要問使用者是否買到真正需要的東西。

以為自己不會受影響 校正

誘餌效應影響的是比較結構，不是只有不懂產品的人才會中。

Questions

把陪襯選項先拿掉

拿掉誘餌後我還會這樣選嗎？

移除那個明顯較差的選項，再看原本兩個方案的偏好是否改變。

這個選項有獨立使用者嗎？

它是否能服務一群真實需求，還是只是在幫另一個方案顯得划算？

延伸路線

可接著讀 compromise effect，對照中間選項如何靠位置感取得安全感。
