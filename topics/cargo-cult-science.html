Orientation

貨物崇拜科學 不像一句立場聲明，它更像討論時的排序器：先把 形式模仿 放回桌面，再決定 誠實檢驗 和 方法實質 要怎麼分配。

Core Tension

模仿了科學的外觀、儀式與術語，不代表你真的保留了會讓自己被打臉的誠實結構。

01 先把 形式模仿 拉回起點

很多爭論表面在談結論，其實一開始就把 形式模仿 當成默認前提。

02 再問 誠實檢驗 是誰在付

現場最常漏掉的不是立場，而是 誠實檢驗 到底由誰承擔、何時結帳。

03 最後確認 方法實質 誰能檢查

真正把事情做完的，往往不是表態，而是把 方法實質 翻成可被檢查的動作。

Setup

哪個前提一旦不寫，貨物崇拜科學 就會被講歪

把討論縮到一次具體決策，形式模仿 是否先被交代、誠實檢驗 是否被偷省、方法實質 是否有驗收方式，就會一眼看出來。

最小劇情

你有報表、有流程、有實驗詞彙，但不願意做會真正暴露問題的對照、重現與反證，最後只剩形式完整。

真正難點

Key Point

把 貨物崇拜科學 用好，重點不在記住一句格言，而在於知道 形式模仿 要先說、誠實檢驗 要有人扛、方法實質 要能驗收。

Lenses

先把 像科學不等於是科學、儀式取代誠實 與 最難的是別騙自己 分開看

如果不把 像科學不等於是科學、儀式取代誠實 與 最難的是別騙自己 拆開，你很容易以為大家在討論同一條原則，實際上是在搶不同的判準。

像科學不等於是科學

順著 像科學不等於是科學 讀，你會發現爭點不在口號，而在 形式模仿 是否真的被變成判準。

儀式取代誠實

把焦點放在 儀式取代誠實 時，你其實是在說：誠實檢驗 應該先被寫清楚，否則後面的結論都站不穩。

最難的是別騙自己

最難的是別騙自己 這條線在乎的，不是誰比較有氣勢，而是 方法實質 一旦沒被交代，現場會怎麼開始失真。

Psychology

為什麼人一忙起來就最容易把 貨物崇拜科學 用歪

這類原則常被用歪，不是因為內容太難，而是因為把它說成一句漂亮話，比真的交代 形式模仿、誠實檢驗 和 方法實質 容易太多。

外觀能快速建立權威

碰到 外觀能快速建立權威 時，最危險的是大家以為原則已經被引用，實際上 形式模仿 還沒被具體說明。

承認不確定太傷

承認不確定太傷 會讓人覺得只要把話說得夠完整，就不必再補 誠實檢驗 的證據或代價。

儀式感比真檢驗容易

儀式感比真檢驗容易 常不是純粹錯誤，而是一種討論捷徑：先把自己放在合理的一邊，再晚點處理 方法實質。

一條原則會變空話，通常不是因為字太抽象，而是因為 方法實質 從來沒有被翻成現場動作。

Applications

回到 A/B 測試品質、數據文化 與 研究流程治理

真正有差的，不是會議上有沒有人引用 貨物崇拜科學，而是 A/B 測試品質 和 數據文化 裡的決策流程有沒有因此變得更可檢查。

A/B 測試品質

在 A/B 測試品質 這種場景，貨物崇拜科學 幫你的不是贏辯論，而是把 形式模仿 先釘在桌面上，避免事後才補。

數據文化

數據文化 最需要 貨物崇拜科學 的時候，通常就是大家都急著往前走，卻還沒把 誠實檢驗 寫成共識。

研究流程治理

只要到了 研究流程治理，你就能看出 方法實質 究竟是被真的承擔，還是只是被口頭提過。

Misreadings

三個常見誤讀

這類頁面最常被讀歪的地方，不是概念太抽象，而是大家太快把 形式模仿 當成價值宣示，忘了它還得連著 誠實檢驗 和 方法實質 一起成立。

貨物崇拜科學 只是好聽口號 校正

真正難的是把 貨物崇拜科學 翻成一組可檢查的判準，而不是把它當成結論本身。

只要背了 貨物崇拜科學 就能直接套用 校正

原則若不落回 形式模仿、誠實檢驗 和代價分配，通常會在現場變成誤用。

貨物崇拜科學 只適合高層決策 校正

其實從 code review 到 方法與誠實 的日常判斷，都會碰到同樣的壓力。
