資源詛咒

Resource Curse

資源詛咒指出天然資源豐富未必帶來繁榮；容易取得的租金可能削弱制度、產業多樣性和公民問責。

Orientation

錢太容易來時，制度可能先學會分錢，而不是創造能力

資源帶來財富，但也可能讓政治和組織圍繞租金分配運轉，降低建立稅收問責、產業能力和長期投資的壓力。

Core Tension

資源收入能資助公共建設，也能養出尋租、貪腐、匯率升值和單一產業依賴。

01 看收入來源

財政若主要來自資源租金，政府與公民的問責關係可能變弱。

02 防止單一依賴

資源繁榮會擠壓製造、服務和知識產業的競爭力。

03 設計透明分配

基金、規則和審計可減少短期政治瓜分。

Setup

一個團隊拿到巨大內部預算後，反而停止磨練產品能力

最小場景是一筆容易取得的資源改變組織行為。

最小劇情

公司把戰略預算固定撥給某平台團隊，不需爭取使用者採用。幾季後，團隊花更多時間守住預算和擴張人力，較少驗證產品是否真的解決問題。

真正難點

資源本身是好事；風險在於它讓回饋和問責被繞過。

Key Point

資源詛咒不是窮比富好，而是提醒容易來的收入需要更強的制度把它轉成能力。

Lenses

看資源是否弱化了回饋迴路

資源越集中、越容易分配，越需要透明和競爭性的治理。

租金來源

收入是否來自稀缺資源控制，而非持續創造價值？

問責鏈

受益者是否需要回應納稅者、用戶或市場？

能力投資

資源是否被用於多樣化、教育、基礎設施和長期韌性？

Psychology

突然富裕會讓短期分配比長期建設更有吸引力

高租金讓爭奪控制權變得非常值得，於是制度注意力轉向誰拿到錢。

尋租收益高

爭取配額可能比提升生產力更划算。

痛苦延後

資源價格高時，制度缺陷不容易立刻顯現。

成功歸因錯誤

收入上升可能被誤認成管理能力提升。

資源越豐厚，越要強化外部審計、分散投資和使用者回饋。

Applications

把租金轉成能力，而不是永久補貼

國家財政、企業平台和基金配置都會面對資源治理問題。

公共財政

主權基金和透明預算可平滑資源價格波動。

企業平台

內部壟斷平台要用服務品質和採用率接受評估。

研發資金

長期補助需搭配里程碑、開放評審和失敗退出機制。

Misreadings

資源詛咒不是說資源本身有毒

真正的關鍵是制度是否能讓資源收入變成公共能力，而非尋租競賽。

把自然資源當原因全部 校正

制度、歷史和國際市場同樣重要。

忽略成功案例 校正

透明治理和長期基金能降低風險。

只套用國家層級 校正

組織內部的壟斷資源也會產生類似誘因。

Questions

問容易來的錢有沒有削弱學習

收入是否繞過了使用者或公民回饋？

若是，需要額外問責機制。

這筆資源在培養能力，還是在維持依賴？

區分一次性投資和永久補貼。
